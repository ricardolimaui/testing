What is Bower?¶

A package manager for the web (Source: http://bower.io, 2016-02-15)

Linux/MacOS¶
  • You need Nodejs to install Bower.
  • Install Bower globally with the following command:
sudo npm install bower -g
Windows¶
  • You need Nodejs to install Bower.
  • Install Bower globally with the following command:
npm install bower -g

Install packages¶

Install all dependecies (bower-packages) of a bower.json-file.

Init¶

bower init

-> bower.json

mkdir lib

-> /public/lib

nano .bowerrc

Bowerrc-File:

{
    "directory":"lib"
}

bower install jquery --save
bower install bootstrap --save
